# Heart Disease Prediction Using Machine Learning

## Project Description

The objective of this project is to build a Machine Learning model that can predict the presence of heart disease based on a patient's medical information. The project uses a real-world dataset obtained from Kaggle and applies various Machine Learning techniques to analyze and predict outcomes accurately.

The project begins with data exploration and preprocessing, where the dataset is checked for missing values, duplicate records, and data consistency. Feature engineering techniques such as correlation analysis and feature importance are used to identify the variables that have the greatest impact on heart disease prediction.

Three different classification algorithms were implemented and compared:

1. Logistic Regression
2. Random Forest Classifier
3. K-Nearest Neighbors (KNN)

The dataset was divided into training and testing sets using an 80:20 ratio. Each model was evaluated using Accuracy, Precision, Recall, and F1-Score to determine its performance. A confusion matrix was also generated for the best-performing model to visualize its prediction results.

After comparing all models, the algorithm with the highest performance was selected as the final model. This project demonstrates the practical application of Machine Learning in healthcare and shows how predictive models can help in the early detection of heart disease.
